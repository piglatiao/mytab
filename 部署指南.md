# 🚀 Cloudflare Pages 部署指南（中文版）

## 📦 最简单的部署方法（5分钟完成）

### 第一步：准备文件

1. 将 `index_new.html` 重命名为 `index.html`
2. 就这一个文件就够了！

### 第二步：注册 Cloudflare

1. 访问：https://dash.cloudflare.com/sign-up
2. 使用邮箱注册（完全免费）
3. 验证邮箱

### 第三步：创建 Pages 项目

1. 登录后，点击左侧菜单 **Workers & Pages**
2. 点击右上角蓝色按钮 **Create application**
3. 选择 **Pages** 标签
4. 点击 **Upload assets** （上传资源）

### 第四步：上传文件

1. **项目名称**：输入一个名字（例如：my-nav）
   - 这个名字会成为您的网址：`https://my-nav.pages.dev`
   
2. **上传文件**：
   - 直接拖放 `index.html` 到上传区域
   - 或点击"选择文件"按钮选择文件

3. **点击 Deploy site**（部署网站）

### 第五步：完成！

- 等待 10-30 秒部署完成
- 您会看到成功页面
- 点击 **Continue to project** 查看您的网站
- 您的网址：`https://项目名.pages.dev`

## 🔄 更新网站

### 方法一：重新上传（简单）

1. 修改本地的 `index.html`
2. 回到 Cloudflare Pages 项目页面
3. 点击 **Create new deployment**
4. 上传新的 `index.html`
5. 完成！

### 方法二：使用 Git（推荐）

如果您会用 Git，可以连接 GitHub 实现自动部署：

1. **创建 GitHub 仓库**
   ```bash
   git init
   git add index.html
   git commit -m "Initial commit"
   git remote add origin https://github.com/你的用户名/仓库名.git
   git push -u origin main
   ```

2. **连接到 Cloudflare**
   - 在 Cloudflare Pages 选择 **Connect to Git**
   - 授权 GitHub
   - 选择仓库
   - 点击 **Save and Deploy**

3. **自动部署**
   - 以后每次 `git push` 都会自动部署
   - 非常方便！

## 🎨 自定义域名（可选）

如果您有自己的域名：

1. 在 Cloudflare Pages 项目中
2. 点击 **Custom domains**
3. 点击 **Set up a custom domain**
4. 输入您的域名（例如：nav.yourdomain.com）
5. 按照提示配置 DNS
6. 等待几分钟生效

## 🔒 安全设置

### 启用 HTTPS（自动）
- Cloudflare 自动提供免费 SSL 证书
- 您的网站自动使用 HTTPS
- 无需任何配置

### 访问控制（可选）
如果想限制访问：
1. 在项目设置中
2. 启用 **Access** 功能
3. 设置密码或邮箱验证

## 💡 常见问题

### Q: 部署后看不到网站？
**A:** 
- 等待 1-2 分钟，DNS 需要时间传播
- 清除浏览器缓存
- 尝试无痕模式访问

### Q: 可以免费使用吗？
**A:** 
- 完全免费！
- Cloudflare Pages 免费版包括：
  - 无限带宽
  - 每月 500 次构建
  - 无限网站
  - 免费 SSL 证书

### Q: 数据保存在哪里？
**A:** 
- 书签数据保存在浏览器的 LocalStorage
- 如果启用云同步，数据会保存到 JSONBin.io
- 网站本身托管在 Cloudflare 的全球 CDN

### Q: 如何备份数据？
**A:** 
- 点击"导出数据"按钮
- 保存 JSON 文件
- 可以随时导入恢复

### Q: 可以在手机上使用吗？
**A:** 
- 完全支持！
- 响应式设计，自动适配手机
- 可以添加到主屏幕当作 App 使用

## 📱 添加到手机主屏幕

### iOS (iPhone/iPad)
1. 在 Safari 打开您的导航页面
2. 点击底部的"分享"图标
3. 向下滚动，选择"添加到主屏幕"
4. 输入名称，点击"添加"
5. 完成！图标会出现在主屏幕

### Android
1. 在 Chrome 打开您的导航页面
2. 点击右上角菜单（三个点）
3. 选择"添加到主屏幕"
4. 输入名称，点击"添加"
5. 完成！
