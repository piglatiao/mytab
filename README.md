## 📖 使用说明

### 添加书签
1. 点击分类旁的"添加网站"按钮
2. 输入网站标题和 URL
3. 自动保存并同步（如果启用）

### 上传背景图片
1. 点击右上角设置图标
2. 选择"上传本地图片"
3. 选择图片文件（支持 JPG、PNG、GIF）
4. 图片会自动转换为 Base64 并保存

### 云同步设置
1. 打开设置 → 云同步
2. 输入一个唯一的同步密钥（建议使用复杂字符串）
3. 启用自动同步
4. 点击"上传数据"保存到云端
5. 在其他设备上使用相同密钥"下载数据"

### 密码保护
- "学习资源"分类默认需要密码
- 默认密码：`123456`
- 点击分类时会提示输入密码

## 🎯 功能特色

### 智能文字背景
所有文字都自动添加半透明背景，确保在任何壁纸下都清晰可读：
- 标题文字：黑色半透明背景 + 文字阴影
- 卡片内容：毛玻璃效果
- 侧边栏：半透明深色背景

### 背景图片管理
- 支持 URL 和本地上传两种方式
- 自动生成缩略图
- 一键切换已上传的背景
- 支持删除不需要的背景

### 数据同步
- 使用 JSONBin.io 免费 API
- 支持自动同步和手动同步
- 同步内容包括：书签、设置、上传的背景
- 使用同步密钥保护数据安全

## 🔧 自定义

### 修改默认密码
在代码中找到 `unlock()` 方法，修改 `correctPassword` 变量：
```javascript
const correctPassword = '你的新密码';
```

### 添加默认书签
在 `loadDefaultBookmarks()` 方法中添加您的书签。

### 自定义主题颜色
修改 Tailwind 配置中的颜色值。
